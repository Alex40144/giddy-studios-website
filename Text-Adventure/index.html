GIDDY STUDIOS

Text Adventure Game

Invasive aliens have landed on your planet. Your job is to rid the planet of aliens and turn on the master switch.

Overview

The main goal of this project was to learn about object oriented programming and classes in Python. To do this I found an article by Phillip Johnson (found here), which walked through how he had designed the game, and explained what every line of his code did. I found this really helpful in making this game. I started with his code and then modified it for my needs. The story line follows that you have to evict an alien species from your planet before they take over.

Walk through

The code is split up in to multiple files, mainly to make navigating the code much easier as we can define the map in one file, the enemies in another, and the main game loop elsewhere. This helps me understand the code better as long files just get confusing. 

while player.is_alive() and not player.victory:
                    room = world.get_tile(player.location_x, player.location_y)
                    room.modify_player(player)
                    if player.is_alive() and not player.victory:
                        time.sleep(2)
                        print("\n===========================================\nChoose an action:\n")
                        available_actions = room.available_actions()
                        for action in available_actions:
                            print(action)
                        action_input = input('Action: ')
                        print("\n\n\n\n\n\n\n")
                        for action in available_actions:
                            if action_input == action.hotkey:
                                player.do_action(action, **action.kwargs)
                                break

This is the main loop of the game, the conditions for the game to continue are that the player is alive and the player has not yet won. If these conditions are met then we can get the tile that the player is on, and run the function of the tile, either giving the player an object or giving the player an enemy to face off against. we are then going to check that the player is still alive and that they have not won the game. The last part of a round is to give the player the curret options for that room, this could be attacking the enemy, or moving on to another room. 

class Player():
                    def __init__(self):
                        self.inventory = [items.Rock()]
                        self.hp = 100
                        self.location_x, self.location_y = world.starting_position
                        self.victory = False

This is the player class, it defines the attributes of the player: Inventory, health points, Location, and victory satus. 

class Item():
                    def __init__(self, name, description, found):
                        self.name = name
                        self.description = description
                
                class Weapon(Item):
                    def __init__(self, name, description, found, damage):
                        self.damage = damage
                        super().__init__(name, description, found)
                
                class Food(Item):
                    def __init__(self, name, description, found, value):
                        self.value = value
                        super().__init__(name, description, found)

This is the item class. Items are objects that can go into the player's inventory. The weapon class is based on the item class, adding an attribute for the value of damage the weapon can do. The food class is also based upon the item class, adding the atttribute for food value. 

class Enemy:
                    def __init__(self, name, hp, damage):
                        self.name = name
                        self.hp = hp
                        self.damage = damage

This is the enemy class, it holds all the attributes that an enemy would have. 

class Bob(Enemy):
                    def __init__(self):
                        super().__init__(name="Bob", hp = 30, damage=5)

This is Bob, one of our enemies. Inside Bob we have defined his name, health points and his damage rating.
